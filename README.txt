These 4 SQL queries investigate the Sakila DVD Rental database. 
It holds information about a company that rents movie DVDs. 
This is the accepted solution of project 2 of the 
"Programming for Data Science with Python" NanoDegree.
Postgres was used as database.
Query 1: Query to analyze the length of rental per genre
Query 2: Customer behavior in late returns, case ALL
Query 3: Customer behavior in late returns, case Sports
Query 4: Customer behavior in late returns, case Travel
Query 5: Seller 1’s performance on a typical weekend
Query 5: Seller 2’s performance on a typical weekend

attached powerpoint.
